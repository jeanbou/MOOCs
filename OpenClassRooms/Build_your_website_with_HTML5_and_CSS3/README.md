# Build your website with HTML5 and CSS3

*About this old course:*

Are you interested in making websites? Look no further! In this course, you will learn how to use HTML5 and CSS3, the two programming languages that all websites are based on.

You might think that you’re not made to learn a computer language, but don’t worry : HTML and CSS are simple languages. The latest versions of these languages (HTML5 and CSS3) now offer us amazing possibilities. With them, you will be able to add text to a website, build a navigation menu, add images and graphic effects, and so much more.

Ready to make a website from scratch? Enroll now!
